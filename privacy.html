🔒 Privacy Policy

Last updated: March 22, 2026

TL;DR: YTNavigator stores all data locally in your browser. We do not collect, store, or transmit any personal information to our servers. Your YouTube API key and all analytics data remain on your device.

1. Introduction

YTNavigator ("we", "us", "our") is a client-side web application that provides YouTube channel and video analytics. This Privacy Policy explains how we handle information when you use our service hosted on GitHub Pages.

2. Information We Collect

We do not collect any personal information. Specifically:

  • No server-side data collection: YTNavigator runs entirely in your browser. We do not operate any backend servers that collect or process your data.
  • No cookies for tracking: We do not use cookies for user tracking, analytics, or advertising purposes.
  • No third-party analytics: We do not use Google Analytics, Facebook Pixel, or any other third-party analytics or tracking services.
  • No user accounts: We do not require registration or user accounts.

3. Data Stored Locally

YTNavigator stores the following data in your browser's Local Storage:

  • YouTube API Key: Your YouTube Data API v3 key, used to make requests to Google's YouTube API.
  • Channel Data: Information about YouTube channels you track (public data fetched from YouTube API).
  • Video Data: Information about YouTube videos (public data fetched from YouTube API).
  • User Preferences: Theme, accent color, notification settings, and other UI preferences.
  • Favorites & Watch Later: Lists of channels and videos you've saved.
  • Alert Configurations: Alert rules you've created.
  • Historical Snapshots: Growth data captured over time for tracked channels.

All this data is stored exclusively on your device and never transmitted to us.

4. Third-Party Services

YTNavigator interacts with the following third-party services:

  • YouTube Data API v3 (Google): When you provide an API key, the application makes requests directly from your browser to Google's YouTube API. These requests are governed by Google's Privacy Policy and the YouTube API Terms of Service.
  • GitHub Pages: This site is hosted on GitHub Pages. GitHub may collect basic server logs (IP addresses, browser user-agent). See GitHub's Privacy Statement.
  • Google Fonts: We use Google Fonts for typography. See Google's Privacy Policy.
  • jsDelivr CDN: We load Chart.js and other libraries from jsDelivr. See jsDelivr's Privacy Policy.

5. Your API Key Security

  • Your YouTube API key is stored in your browser's Local Storage.
  • The key is used only to make direct requests to Google's YouTube Data API from your browser.
  • We never transmit your API key to any server operated by us.
  • We recommend restricting your API key in the Google Cloud Console to only the YouTube Data API v3 and to HTTP referrers matching the domain where you host YTNavigator.

6. Data Retention & Deletion

  • All data is stored in your browser's Local Storage and persists until you clear it.
  • You can export all your data at any time from Settings → Export All Data.
  • You can delete all data at any time from Settings → Clear All Data.
  • Clearing your browser's Local Storage or site data will also remove all YTNavigator data.

7. Children's Privacy

YTNavigator is not directed at children under 13. We do not knowingly collect information from children. If you believe a child has provided data through this application, please clear the browser's Local Storage.

8. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected on this page with an updated "Last updated" date. Continued use of YTNavigator after changes constitutes acceptance of the updated policy.

9. Open Source

YTNavigator is open source. You can inspect the complete source code to verify our privacy practices. The source code is the ultimate proof of what the application does and does not do.

10. Contact

If you have questions about this Privacy Policy, please open an issue on our GitHub repository.
